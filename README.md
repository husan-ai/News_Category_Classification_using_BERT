# BERT-based News Classification System

A simple NLP project that classifies news articles into 4 categories using a fine-tuned BERT model.

## Features
- Fine-tuned **bert-base-uncased**
- Classifies news into:
  - World
  - Sports
  - Business
  - Sci/Tech
- Built with **Hugging Face Transformers**
- Interactive UI using **Gradio**

## Dataset
- AG News dataset
- 2000 training samples
- 400 test samples

## Tech Stack
- Python
- PyTorch
- Transformers
- Datasets
- Gradio
